## What is DevRadar?

DevRadar helps developers:

- **Check Compatibility** - Validate if Next.js works with Prisma, if Supabase supports Vercel Edge, and more
- **Compare AI Tools** - Lovable vs Bolt.new vs Cursor vs Claude Code - find the best AI coding tool
- **Avoid Pitfalls** - Known issues, workarounds, and version-specific conflicts

## Data Sources

Our compatibility database is built from:

- Official documentation and release notes
- GitHub repositories and issue trackers
- Developer community (Stack Overflow, Reddit, Discord)
- Direct testing for critical compatibility rules

## Categories Covered

- **Frameworks** - Next.js, Nuxt, SvelteKit, Remix, Astro, Hono
- **Databases** - PostgreSQL, MongoDB, Supabase, Firebase, Turso, PlanetScale
- **Hosting** - Vercel, Netlify, Cloudflare, Railway, Render, Coolify
- **Authentication** - Clerk, Auth0, NextAuth, Supabase Auth, Firebase Auth
- **AI Tools** - Lovable, Bolt.new, Cursor, Claude Code, Windsurf, Replit Agent
- **Utilities** - ORMs, validation libraries, payment processors
